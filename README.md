## Playground

##### 1. Pull the repository

```bash
git clone https://github.com/schodevio/qbricks-splitpanes.git
```

##### 2. Install packages

Install all required dependencies.

```bash
yarn install
```

##### 3. Run dev server

```bash
yarn serve
```

##### 3. Run final build

Run build to get final version.

```bash
yarn build
```
